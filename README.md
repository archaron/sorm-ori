# СОРМ ОРИ
Приказ Минкомсвязи России от 29.10.2018 N 571 "Об утверждении Требований к оборудованию и программно-техническим средствам, используемым организатором распространения информации в сети "Интернет" в эксплуатируемых им информационных системах, для проведения уполномоченными государственными органами, осуществляющими оперативно-розыскную деятельность или обеспечение безопасности Российской Федерации, мероприятий в целях реализации возложенных на них задач" (Зарегистрировано в Минюсте России 27.12.2018 N 53207)
